## AI-Powered Elasticsearch

## 概述

本项目是参照AI-Powered Search[^1]书籍和Elasticsearch Learning to Rank[^2]指南构建的示例应用，以期望实现智能化搜索。


## 功能
- [ ] Signal Boosting
- [ ] Auto Learning to Rank
- [ ] Query Suggestion
- [ ] 知识图谱
